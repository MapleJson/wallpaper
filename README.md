# wallpaper

#### 示例

[https://wallpaper.52zoe.com](https://wallpaper.52zoe.com)

> ThinkPHP5的运行环境要求PHP5.6以上。

## 安装

使用composer安装

~~~
git clone https://github.com/MapleJson/wallpaper.git

cd wallpaper

composer install
~~~

启动服务

~~~
cd wallpaper

php think run
~~~

然后就可以在浏览器中访问

~~~
http://localhost:8000
~~~

更新框架
~~~
composer update topthink/framework
~~~
